    k-search-input-text es un componente que permite ingresar un valor y las respectivas condiciones para la aplicación de filtros.

    Importación:

    En el componente donde se va a utilizar:

    TS
    ?
    @Component({
      selector: 'app-persons-filters',
      standalone: true,
      imports: [
        CommonModule,
        FormsModule,
        KSearchInputTextComponent
      ],
      templateUrl: './persons-filters.component.html',
      styleUrl: './persons-filters.component.scss'
    })
    export class PersonsFiltersComponent {
        constructor() {}
    }

    Ejemplo de uso

    home.component.html
    ?
    <k-search-input-text id="processName" name="processName"
                       [(ngModel)]="searchFilter.processName"
                       [label]="'Process name:'"
                       [optionSettings]="optionSettingsAll"
                       [placeholder]="labels.processName"
                       [maxlength]="'10'" (searchOnEnter)="clickSearch()"
                       [upperInputValue]="true" [regularExpression]="'[a-zñA-ZÑ 0-9]'" [required]="true"
                      [dispatch]="dispatchEvent">
    </k-search-input-text>

    Atributos

    Propiedad
    Requerido
    Tipo
    Valor por defecto
    Descripción
    ngModelSiSearchModel

    Modelo contenedor de los atributos para la búsqueda.

    labelNoString

    Etiqueta del componente.

    optionSettingsNoOptionConfigModel{SHOW_ERASER: true, SHOW_LIST_VALUES: true, SHOW_OPERATOR: true}

    Define si se mostrarán los botones de opciones: operador, configuraciones y borrador.

    requiredNoBooleanfalse

    Especifica que el campo debe ser llenado de manera obligatoria.

    nameSiString

    Nombre del componente.

    dispatchNoDispatchEvent

    Recibe eventos para modificar el componente (Los eventos que puede recibir son los incluidos en KSearchDispatch).

    cleanNoEventEmitter

    Evento que es emitido cuando se borra el contenido del campo.

    placeholderNoString

    Texto a mostrar cuando el campo de entrada es vacío.

    showComparatorTypeNoBooleantrue

    Permite mostrar u ocultar el selector de los comparadores como: Inicio, final, Contenga.

    inputStyleClassNoString

    Asigna una clase de estilo existente al componente.

    searchOnEnterNoEventEmitter

    Evento que es emitido cuando en el enfoque del componente se presiona la tecla enter.

    focusNoEventEmitter

    Evento que es emitido cuando el campo de entrada tiene el enfoque.

    blurNoEventEmitter

    Evento que es emitido cuando el campo de entrada pierde el enfoque.

    inputNoEventEmitter

    Evento que es emitido cuando el campo de entrada recibe un valor, es utilizado internamente por el componente para verificar los caracteres ingresados sean válidos.

    regularExpressionNoString

    Expresión regular para validar caracteres permitidos.

    disabledCommaListNoBooleanfalse

    Deshabilita que el caracter , añada un elemento como lista o cambie al modo Lista de Valores.

    patternExpressionNoString

    Valida que en el campo de entrada sólo exista la expresión regular indicada (no bloquea el ingreso de caracteres, pero indica el campo no es válido).

    pasteNoEventEmitter

    Evento que es emitido cuando el campo de entrada recibe datos de la función pegar.

    allowRepeatedItemsNoBooleantrue

    Permite ingresar valores repetidos en el modo Lista de valores.

    maxlengthSiNumber

    Cantidad máxima de caracteres que admite el campo de entrada.

    upperInputValueSiBooleanfalse

    Transforma el contenido del campo de entrada en mayúsculas.
